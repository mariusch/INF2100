# Pascal 2016 compiler
> Partial Pascal compiler written in Java in the course INF2100 at the University of Oslo fall 2016

## Run and usage

OS X, Linux & Windows:

```sh
git clone <repo>
ant build

Usage: java -jar pascal2016.jar " +
            "[-log{B|P|S|T|Y}] [-test{checker|parser|scanner}] file
```
